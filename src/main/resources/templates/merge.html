Commentaire sur la maquette

Ici on a l'équivalent du merge. Comme pour la préparation d'un lot, les modifications sont spécifiées dans le tableau mais appliquées qu'avec la confirmation finale. Pour la résolution de conflit, dans une popin on affiche la ligne correspondante dans la BDD cible (courante), celle reçue dans le lot importé, et on peut saisir directement la valeur utilisée dans un texte field. Il peut être nécessaire de raffraichir le diff local en arrivant. Le bouton refresh permet de relancer cela
